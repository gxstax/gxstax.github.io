yum 基本命令

1. yum ：参数帮助 类似help
2. yum list： 列出所有中央仓库软件（一般不要这样用要加过滤条件）
3. yum list | grep jdk：列出带有关键子 jdk的软件信息
            ps:使用yum软件的好处在于，如果我们用传统的解压包方式，如果A软件依赖B软件，我们要
                  先安装了B然后在安装A，如果依赖的比较多的化就会比较繁琐，使用yum的它会自动帮我
                  们依赖好，是不是很方便。
4. yum install  xxx：安装某个软件
5. yum install xxx -y： 免提示安装
6. yum list updates ：列出所有可更新的软件包 
7. yum list installed ：列出所有已安装的软件包 
8. yum list extras ：列出所有已安装但不在 Yum Repository 内的软件包 
9. yum info : 使用YUM获取软件包信息 
10.  yum info :  列出所有软件包的信息
11. yum info updates : 列出所有可更新的软件包信息
12. yum info installed : 列出所有已安装的软件包信息
13. yum info extras : 列出所有已安装但不在 Yum Repository 内的软件包信息
14. yum provides : 列出软件包提供哪些文件
15. yum -y remove xxxx : 卸载一个软件时
16. yum -y remove httpd* : 卸载多个相类似的软件时
17. yum -y remove httpd php php-gd mysql : 卸载多个非类似软件时

附带写点解压命令：

tar -zxvf xxxx.tar.gz ： 解压某个压缩包
    z : 表示 tar 包是被 gzip 压缩过的，所以解压时需要用 gunzip 解压
    x : 从 tar 包中把文件提取出来
    v : 显示详细信息
    f ：xxx.tar.gz : 指定被处理的文件是 xxx.tar.gz
tar -zxvf  xxxx.tar.gz -C /xxx/xxx :  解压到指定的路径下
